# v0.5.0 ‚Äî Honest Evolution

**Date:** 2026-05-30

## Philosophy

"Honest evolution" ‚Äî replace LLM self-evaluation with framework-measured behavior
metrics. Quality gates S1/S4 are labeled "no LLM participation". The emergence
verifier uses zero LLM calls. Evolution is judged by tool success rate, action
diversity, and drive intensity ‚Äî not by asking the LLM to grade itself.
